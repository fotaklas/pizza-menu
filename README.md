# Pizza Menu (React Learning Project)

Ένα μικρό project για εξάσκηση στο **React**.  
Δεν είναι production app — είναι καθαρά για να μαθαίνω βασικές έννοιες.

## Τι κάνει

Η εφαρμογή δείχνει ένα απλό μενού πίτσας:

- λίστα από πίτσες (όνομα, υλικά, τιμή, εικόνα)
- ένδειξη **SOLD OUT** όταν μια πίτσα δεν είναι διαθέσιμη
- απλό footer που δείχνει αν το μαγαζί είναι ανοιχτό ή κλειστό ανάλογα με την ώρα

## Τι εξασκώ εδώ

- Function components
- Props
- Rendering λίστας με `map`
- Conditional rendering
- Βασικό styling με CSS

## Τεχνολογίες

- React
- Create React App
- CSS

## Scripts

Στον φάκελο του project:

- `npm start` → τρέχει το app σε development mode
- `npm test` → ανοίγει test runner
- `npm run build` → production build
- `npm run eject` → eject (μη αναστρέψιμο)

## Σημείωση

Αυτό το project είναι εκπαιδευτικό και φτιαγμένο για practice στο React.
